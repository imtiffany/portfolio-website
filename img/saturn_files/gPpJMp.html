Let's try to make a good and complete demo for SVG on the web :)

path, line, polygon, rect circle, and ellipse are HTML tags.

fill and stroke are CSS properties.

SVG Composition

Moon is composed of circle's with stroke.
Saturn is a path (a filled shape). You can't change line thickness.
Hover to see the outline view.

Colors on SVG

Colors can be set in the SVG or in CSS (for inline SVG only).
Rain has inline styles and flash has CSS styles.

SVG stroke styles

The first flash use default values (stroke-linecap: butt and stroke-linejoin: miter).
The second flash use stroke-linecap: square and stroke-linejoin: bevel.
The third flash use stroke-linecap: round and stroke-linejoin: round.

Sprite SVG

The moon is called with a use tag.
Call a svg with a CSS background has the same contraints then call an external SVG (you can't change color, etc.).
